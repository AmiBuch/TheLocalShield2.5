# Assets Directory

This directory should contain the following files for your Expo app:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (foreground image)
- `favicon.png` - Web favicon

## Quick Setup

For development, you can use placeholder images or generate them using:

1. **Online tools:**
   - [App Icon Generator](https://www.appicon.co/)
   - [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)

2. **Or create simple placeholder images:**
   - Create a 1024x1024px PNG for `icon.png`
   - Create a 1284x2778px PNG for `splash.png` (iPhone 14 Pro Max size)
   - Create a 1024x1024px PNG for `adaptive-icon.png`
   - Create a 48x48px PNG for `favicon.png`

For now, the app will work without these assets, but you'll see warnings. You can add them later.
